Algebra Notes

1. What is Algebra?

Algebra involves using symbols (like x or y) to represent numbers in equations and expressions. It's the foundation of higher-level math.

2. Basic Concepts

  • Variables: Letters that stand for unknown values (e.g., x, y)
  • Constants: Fixed values like 2, -5, or 3.14
  • Expressions: Combinations of variables, constants, and operations (e.g., 2x + 5)
  • Equations: Expressions set equal to one another (e.g., 2x + 5 = 15)

3. Solving Equations

  • Isolate the variable
  • Perform inverse operations
  • Check your solution by plugging it back in

4. Types of Algebraic Equations

  • Linear equations (e.g., 3x + 2 = 11)
  • Quadratic equations (e.g., x² + 5x + 6 = 0)
  • Systems of equations

5. Factoring Techniques

  • Common factoring
  • Difference of squares: a² - b² = (a - b)(a + b)
  • Trinomials: x² + bx + c
